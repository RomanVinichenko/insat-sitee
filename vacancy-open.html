Angular Front-end Senior Developer

We are looking for experienced professionals capable of designing and developing complex systems using Angular, working out technical solutions and promoting development best-practices.

Poland Office, Remote

Requirements

  • 3+ years commercial experience in developing single-page applications (SPA) using Angular
  • Experience with Reactive Programming in Angular
  • Confident command of HTML5 and CSS3, TypeScript and JavaScript
  • Experience with style preprocessors (Sass, Less, Stylus)
  • Experience in writing automated tests using standard and external tools (Protractor, TestBed, Cypress)
  • Knowledge of SOLID, KISS, DRY principles
  • Knowledge of approaches to optimizing the client side of applications
  • Good command of English. Oral communication in English on technical topics

  • Experience in organizing development processes (CI/CD, code review, static code analysis, automated testing)
